This is landing page website and CRUD upload image

## Getting Started

First, go to the backend directory and install the dependencies:

```bash
cd backend
npm install
```

second, go to the frontend directory and install the dependencies:

```bash
cd backend
npm install
```

### Start

for backend

```bash
nodemon index
```

for frontend

```bash
npm start
```
